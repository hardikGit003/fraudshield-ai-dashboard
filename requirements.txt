# 💳 FraudShield AI — Credit Card Fraud Detection Dashboard

An AI-powered fraud detection and analytics dashboard built using Machine Learning, Streamlit, and Python. This project helps detect fraudulent credit card transactions through real-time prediction, live monitoring, and bulk CSV fraud scanning.

---

## 🚀 Features

- 🔍 Real-Time Fraud Prediction
- 📊 Interactive Data Visualizations
- 📈 Model Analytics Dashboard
- 📡 Live Transaction Monitoring
- 📂 Bulk CSV Fraud Scanner
- 🧠 AI-Based Fraud Risk Insights
- 📥 Downloadable Fraud Reports
- 🎨 Modern Streamlit UI Dashboard

## 📈 Machine Learning

- Logistic Regression model used for fraud classification
- Feature engineering and preprocessing applied
- Model evaluation using:
  - ROC Curve
  - Confusion Matrix
  - Precision & Recall
  - Classification Report

## ▶️ Run Locally

### Clone the repository

```bash
git clone https://github.com/hardikGit003/fraudshield-ai-dashboard.git
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Run the Streamlit app

```bash
streamlit run app.py
```
